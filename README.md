# Vision

a cli tool that helps me to focus and reflect my days

## Features

`check in`

- [ ] set my daily goals
- [ ] reflect on yesterday (optional)
- ...

`check out`

- [ ] reflect on daily goals
- [ ] check my habits
- [ ] write a short journal (optional)

`check db` (db can be 'dashboard' or 'database' ^^)

- [ ] see all entries
- [ ] see statistics
